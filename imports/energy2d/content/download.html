Energy2D

Interactive Heat Transfer Simulations for Everyone

Download Energy2D

Energy2D is available for downloading in two formats: Java Webstart and Java executable. Java Webstart is available for Windows & Linux and for Mac OS X. The java executable is for all operating systems.

Although Energy2D is licensed to you under the GNU Lesser General Public License V3.0, the license does not extend to the simulations you have created. Feel free to distribute your simulations in the way you need.
