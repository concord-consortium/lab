Dirichlet Boundary Condition

Defining the Values on the Boundary of the Problem Domain

The Dirichlet boundary condition is the first type of boundary condition that defines the values on the border of the problem domain. In the following simulation, you can set temperatures of different borders and see how this boundary condition will affect the temperature distribution inside the box.
