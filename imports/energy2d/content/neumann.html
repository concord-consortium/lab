Neumann Boundary Condition

Defining the Gradients on the Boundary of the Problem Domain

The Neumann boundary condition is the second type of boundary condition that defines the gradients on the border of the problem domain (the gradient at a spot along the border describes how much the value across the boundary changes). In the following simulation, you can set heat fluxes through different borders and see how this boundary condition will affect the temperature distribution inside the box.
