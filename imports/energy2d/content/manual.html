How to use Energy2D

Energy2D is:

  • a two-dimensional interactive computational fluid dynamics (CFD) and heat transfer simulation program;
  • an inquiry tool students can use to investigate heat flow and energy usage in structures;
  • both a Java application that can be downloaded to run offline and a Java applet that runs directly withina browser;
  • free and open-source.

Javascript controls

Energy2D scripts:

  • View options
    • set clock true/false: Show or hide the digital clock
    • set graph true/false: Show or hide the graph
    • set grid true/false: Show or hide the grid lines
    • set heat_flux_arrow true/false: Show or hide the heat flux arrows
    • set heat_flux_line true/false: Show or hide the heat flux lines
    • set isotherm true/false: Show or hide the isotherm lines
    • set rainbow true/false: Show or hide the rainbow
    • set ruler true/false: Show or hide the rulers
    • set seethrough true/false: Set coloring through the parts
    • set streamline true/false: Show or hide the streamlines
    • set velocity true/false: Show or hide the velocity vectors
